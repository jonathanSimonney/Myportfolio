Technologies

Mes technologies

JS

JS est un langage permettant d'ajouter de l'intéraction pour le front. De nombreuses technologies se basent également sur ce langage, mais il m'a semblé plus judicieux d'en parler comme des technologies à part.

ReactJS

Framework front de JS qui permet de mieux structurer le JS et qui marche sur le principe des composants. Une technologie en étant dérivé est react-native qui permet le développement d'applications hybrides.

VueJS

Autre framework front de JS qui permet de mieux structurer le JS et qui marche sur le principe des composants. Il est plus simple à prendre en main que React, mais demeure moins utilisé car plus récent. Des outils existent également pour permettre d'utiliser vue pour des applications hybrides ou pour des applications desktop.

NodeJS

JS utilisé en back. J'ai pu faire quelques projets avec le framework express. L'avantage de node est qu'il permet d'utiliser le même langage en back et en front.

CSS

Language qui permet de styliser le HTML et de rajouter des animations .

HTML

Language qui permet de présenter le contenu aux navigateurs. (tout les langages back servent au final juste à afficher du html, du css et du JS.) .

PHP

Langage extrêmement utilisé en back office. De nombreux framework php existent, et j'ai pu entre autre travailler avec Symfony, Laravel ou wordpress.

Python

Langage back office. J'ai pu utiliser le framework python du nom de django. Python est également utilisé pour du machine learning (mais je n'ai pas eu l'occasion de l'utiliser dans ce contexte)

ruby

Langage back office. J'ai pu utiliser le framework ruby du nom de rubyOnRails. Ruby est très utile pour sortir un MVP (minimal viable product) rapidement.

golang

Langage back office créé par google. Golang est un langage avec très peu de mots clefs, fortement typé et très optimisé.

java

Langage back office. Java est un langage fortement typé. Il est utilisé pour la programmation sur android et pour du back office avec le framework spring boot. (2 utilisation que j'ai pu faire de ce lanagage.)

swift

Langage créé par apple pour la programmation back office ou le développement IOS. J'ai pu l'utiliser pour des applications IOS.
